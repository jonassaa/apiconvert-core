# Canonical Rules Model

Rules are an ordered recursive tree under one `rules` array.

## Node kinds

- `field`
- `array`
- `branch`
- `use` (fragment expansion)

## Supported formats

- `json`
- `xml`
- `query`

## Supported source types

- `path`
- `constant`
- `transform`
- `merge`
- `condition`
